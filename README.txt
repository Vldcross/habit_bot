
ИНСТРУКЦИЯ ПО ЗАПУСКУ НА RENDER:

1. Создай аккаунт на https://render.com/.
2. Залей папку habit_bot_server на GitHub как репозиторий (или импортируй напрямую через Render CLI).
3. В Render выбери New + -> Web Service -> Python.
4. Укажи команду запуска: ./start.sh
5. Нажми Deploy. Бот будет работать 24/7.

Если что-то не понятно — пиши в чат.
